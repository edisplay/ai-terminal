# Security Policy

## Supported Versions
- Only the `main` branch is actively supported.
- Security fixes are applied to the latest **AI Terminal version**.  
- Users are encouraged to upgrade to the latest release: `v1.x.x` (or latest tag).

## Reporting a Vulnerability
To report a security vulnerability, please use [GitHub Security Advisories](https://github.com/AiTerminalFoundation/ai-terminal/security/advisories/new).

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any relevant logs, screenshots, or proof-of-concept code

## Responsible Disclosure
Please allow time for remediation before any public disclosure.
